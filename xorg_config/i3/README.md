# Prerequisite
* i3-gaps (Arch)
* i3lock-fancy (AUR)
* flameshot (Arch)
* brightnessctl (Arch)
* pamixer (Arch)
* xidlehook (AUR)
* xob (AUR)
* rofi (Arch)
* rofi-power-menu (AUR)
* nitrogen (Arch)
* polybar (Arch)

# Installation

1. Copy xsession file
```
cp xsessionrc $HOME/.xsessionrc
```

2. Copy i3 config file
```
cp config $HOME/.config/i3/config
```
